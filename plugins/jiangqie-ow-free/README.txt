== Description ==

## 功能与特点

特点：多端（微信、QQ、百度、H5）、免费开源、功能后台配置

模块：留言反馈、产品服务、新闻动态、合作伙伴、关于我们等

微信：**jianbing2011**（加微信可微信群交流）

== Installation ==

1. 把jiangqie-ow-free文件夹上传到/wp-content/plugins/目录下

2. 在后台插件列表中启用"酱茄企业官网Free"

3. 设置微信/QQ/百度小程序的appid，及其他配置信息
